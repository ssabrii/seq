class Stmt : public seq::SrcObject¶

Class from which all Seq statements derive.

A “statement” can be thought of as an AST node, potentially containing sub-expressions and sub-statements.

Subclassed by seq::Assert, seq::Assign, seq::AssignIndex, seq::AssignMember, seq::Break, seq::Continue, seq::Del, seq::DelIndex, seq::ExprStmt, seq::For, seq::FuncStmt, seq::If, seq::Match, seq::Prefetch, seq::Print, seq::Return, seq::Throw, seq::TryCatch, seq::VarStmt, seq::While, seq::Yield
